# HML-LiDAR PROTO

This folder contains the current prototype configuration of the HML-LiDAR cave mapping system.

PROTO is the working development version. It documents the field-tested prototype, including hardware, power supply, wiring, ROS2/DLIO configuration files, processing scripts, technical documentation, and sample outputs.

This version is not yet a stable release. Stable hardware/software snapshots will be published as RevA, RevB, etc.
